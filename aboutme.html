CIRCUIT THAT TURNS ON AN LED

This is the first robotics project we made. We used an Arduino, Board of Education shield, breadboard, jumper wires, resistor, LEDs, and a button. I learned which way the power runs and how a circuit has to be compelete. I also learned how everything has to line up in order for all the components to work.
